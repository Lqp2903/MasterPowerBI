# MasterPowerBI
 Curso de Power BI, para entender suas funcionalidades e tratamento de dados.

 No primeiro módulo, foi aprendido as seguintes ferramentas:
 Baixar arquivos Excel e utilizar no PBI. Mas como eu gosto de testar, transformei em arquivo texto, e o PBI entendeu da mesma forma;
✅ Opção "transformar dados" é super importante! Citando o exemplo dessa empresa, havia dados como "ID da venda" que se não fosse transformado em texto, seria somado. Ao transformá-lo em texto, o sinal de sigma não estava mais ali. Além disso, o próprio PBI avisa se há algum erro nos dados baixados;
✅Criar cartões, nesse caso, com o total de vendas, lucro total, valor médio (com a opção média) das vendas e número de produtos vendidos ( com a opção contagem distinta);
✅ Gráfico de área para venda de ano e por trimestre utilizando os dados baixados data da venda e valor da venda;
✅ Gráfico de colunas clusterizado para entender o lucro gerado por categoria utilizando os dados baixados categoria e lucro;
✅ Gráfico de pizza para entender o lucro gerado dos canais de vendas utilizando os dados baixados canal e valor da venda;
✅ Gráfico de dispersão para entender se lucro e venda dos produtos batem. Como o Felipe disse: "Eu posso ter um produto caro vendendo bem, ou um produto barato vendendo bem, mas se ele não bater com o lucro, pode ter algo errado.", utilizei os dados baixados lucro, subcategoria e valor da venda.
✅ E por fim, criei um filtro chamado "segmentação de dados", onde alinhado com os dados baixados "continente", ele me mostra os dados de cada respectivo continente;
✅ Uma observação que acho bacana comentar é que, se eu quiser selecionar um dado que não esteja no filtro, basta eu clicar em algum dado dos gráficos que ele traz automaticamente um dado separado.
